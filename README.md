# Simple Menu
![screenshot](preview_menu.png)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](https://opensource.org/licenses/MIT)

a tiny rust + macroquad toy.  
you get a main menu, a fake game screen, and a settings screen where you can click a button to cycle the background color.

## features

- main menu with a start button  
- game screen with placeholder text  
- settings screen with:
  - info text
  - background color picker (5 colors, cycles on click)

## how to run

```bash
cargo run
